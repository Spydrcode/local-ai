## Additional Frameworks to Implement

### 3. BCG Growth-Share Matrix
**Endpoint:** `/api/strategic-frameworks/bcg-matrix/[demoId]`

**Analysis includes:**
- Stars (high growth, high market share) - invest heavily
- Cash Cows (low growth, high share) - milk for cash
- Question Marks (high growth, low share) - evaluate potential
- Dogs (low growth, low share) - divest or harvest
- Portfolio recommendations

**Use case:** Multi-product/service businesses deciding where to allocate resources

**Implementation priority:** HIGH

---

### 4. McKinsey 7S Framework
**Endpoint:** `/api/strategic-frameworks/mckinsey-7s/[demoId]`

**Analysis includes:**
- Strategy: Plan to gain competitive advantage
- Structure: How organization is organized
- Systems: Daily processes and procedures
- Shared Values: Core values and company culture
- Style: Leadership approach
- Staff: Employees and their capabilities
- Skills: Core competencies
- Alignment analysis across all 7 elements

**Use case:** Organizational change, M&A integration, restructuring

**Implementation priority:** MEDIUM

---

### 5. Competitive Positioning Map
**Endpoint:** `/api/strategic-frameworks/positioning-map/[demoId]`

**Analysis includes:**
- 2x2 matrix plotting competitors on key dimensions
- Common dimensions: Price vs Quality, Service vs Product, Innovation vs Reliability
- Visual identification of market gaps
- Positioning recommendations

**Use case:** Quick visual for understanding competitive landscape

**Implementation priority:** HIGH

---

### 10. Core Competencies Analysis (Prahalad & Hamel)
**Endpoint:** `/api/strategic-frameworks/core-competencies/[demoId]`

**Analysis includes:**
- Identification of core competencies
- Test against 3 criteria: provides customer value, differentiation, extensibility
- Competency-based growth opportunities
- Competencies to build vs outsource
- Competitive moat assessment

**Use case:** Strategic focus, outsourcing decisions, M&A targets

**Implementation priority:** MEDIUM

### 15. OKR Framework (Objectives & Key Results)
**Endpoint:** `/api/strategic-frameworks/okr-framework/[demoId]`

**Analysis includes:**
- Company-level objectives (3-5 aspirational goals)
- Key Results (2-5 measurable outcomes per objective)
- Department/team cascading OKRs
- Progress tracking mechanisms
- Quarterly vs annual OKRs
- Example OKRs for similar businesses

**Use case:** Goal setting, performance management, alignment

**Implementation priority:** HIGH

---

### 16. Scenario Planning
**Endpoint:** `/api/strategic-frameworks/scenario-planning/[demoId]`

**Analysis includes:**
- Key uncertainties affecting business
- 4 distinct future scenarios (2x2 matrix)
- Implications of each scenario
- Strategic options for each scenario
- Early warning indicators
- Hedging strategies

**Use case:** Long-term planning, risk management, strategic flexibility

**Implementation priority:** MEDIUM

---

### 17. Competitive Advantage Period (CAP) Analysis
**Endpoint:** `/api/strategic-frameworks/competitive-advantage-period/[demoId]`

**Analysis includes:**
- Current competitive advantages
- Estimated CAP duration for each advantage
- Erosion threats and speed
- Renewal strategies
- Next-generation advantages to build
- Portfolio of staggered advantages

**Use case:** Sustained competitive advantage, strategic renewal

**Implementation priority:** LOW

---

### 18. Strategic Group Mapping
**Endpoint:** `/api/strategic-frameworks/strategic-groups/[demoId]`

**Analysis includes:**
- Identification of strategic groups in industry
- Key dimensions differentiating groups
- Mobility barriers between groups
- Group profitability analysis
- Opportunities to switch groups
- Competitive dynamics within/between groups

**Use case:** Competitive analysis, positioning decisions

**Implementation priority:** MEDIUM

---

### 19. Stakeholder Analysis & Mapping
**Endpoint:** `/api/strategic-frameworks/stakeholder-analysis/[demoId]`

**Analysis includes:**
- Stakeholder identification (power/interest matrix)
- Stakeholder needs and expectations
- Influence and impact assessment
- Engagement strategies for each stakeholder group
- Coalition-building opportunities
- Risk mitigation for resistant stakeholders

**Use case:** Change management, major initiatives, crisis management

**Implementation priority:** LOW

---

### 20. Digital Maturity Assessment
**Endpoint:** `/api/strategic-frameworks/digital-maturity/[demoId]`

**Analysis includes:**
- Current digital maturity level (1-5 scale)
- Assessment across: technology, data, processes, people, culture

- Industry benchmarking
- Digital transformation roadmap

**Use case:** Digital transformation planning

**Implementation priority:** HIGH

## Framework Selection Guide (For Agencies)

**If client needs...**

### Growth Strategy:
- Ansoff Matrix (which direction to grow)
- BCG Matrix (where to invest)
- Blue Ocean Strategy (how to escape competition)

- Porter's Five Forces (industry attractiveness)
- Competitive Positioning Map (visual differentiation)
- Strategic Group Mapping (who competes how)

- Blue Ocean Strategy (value innovation)
- Jobs-to-be-Done (unmet customer needs)
- Disruptive Innovation (threat/opportunity assessment)

### Business Model:
- Business Model Canvas (holistic view)
- Platform Model Analysis (network effects)
- Value Chain Analysis (where to play, how to win)

### Organizational Change:
- McKinsey 7S (alignment check)
- Stakeholder Analysis (managing change)
- Core Competencies (what to build vs buy)

### Digital Transformation:
- Digital Maturity Assessment (current state)
- Customer Journey Mapping (CX optimization)
- Platform Model Analysis (digital opportunities)

### Strategic Planning:
- SWOT + PESTEL (environmental scan)
- Scenario Planning (future preparedness)
- OKR Framework (goal setting)

## Technical Implementation Pattern

All frameworks follow this pattern:

```typescript
// 1. Validate request
// 2. Fetch demo data
// 3. Check agency report limits
// 4. Generate analysis using AI with structured prompt
// 5. Parse JSON response
// 6. Log activity and increment usage
// 7. Return structured data
```

**Common response format:**

```typescript
{
  success: true,
  data: {
    summary: "Executive summary of analysis",
    sections: {
      // Framework-specific sections
    },
    recommendations: ["Prioritized action items"],
    nextSteps: ["What to do next"],
    resources: ["Links to learn more about framework"]
  }
}
```
